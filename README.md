# UFO Simulator v2

PixiJS v8 で描くインタラクティブ UFO シミュレーター

## 技術スタック

| 技術 | バージョン |
|------|-----------|
| TypeScript | 5.x |
| PixiJS | v8 |
| Vite | 7.3 |

## 主な特徴

- グローエフェクト・パーティクル・軌跡表示
- 物理ベース移動（加速度・摩擦・ブースト）
- Entity-Component 的アーキテクチャ

## 開発

```bash
npm install
npm run dev
```

`http://localhost:3000` で起動します。
